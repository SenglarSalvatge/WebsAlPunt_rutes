Enrut.cat

enrut.cat: Projecte.

Aquest projecte forma part del concurs Webs Al Punt.
El tema triat, és la creació d'una aplicació web que permeti a un usuari crear rutes (Per córrer, per fer a peu, amb bicicleta etc...). A aquestes rutes, s'hi podràn apuntar altres usuaris de l'aplicació. També es podrà comentar una activitat, generar debat entre usuaris, modificar-la etc...

enrut.cat: Creació

L'aplicació està desenvolupada amb el Framework de Python Django (v. 1.6) per a la creació dels models, els formularis d'entrada de dades i la gestió d'usuaris.
El disseny està fet amb Twiter Bootstrap 3.

enrut.cat: Desenvolupadors.

Aplicació creada pels alumnes Álex Ramírez, Darwin López, Martí Peris i Rodrigo Saulis.
Actualment cursant segón de CFGS de Desenvolupament D'Aplicacions Web (DAW) a l'INS Cendrassos de Figueres (17600, Girona).
